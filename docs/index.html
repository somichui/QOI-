How It Works

Three algorithmic improvements over the original QOI specification

2D Prediction

Uses Paeth-style prediction combining the pixel above and to the left, reducing residuals by up to 30% compared to QOI's left-only predictor.

256-Slot Hash Table

Expands the color hash table from 64 to 256 slots, dramatically reducing collisions and improving index-hit rates for complex images.

Huffman Entropy Coding

Optional Huffman pass re-encodes op-code frequencies, squeezing an additional 5–15% from the bitstream without sacrificing decode speed.
