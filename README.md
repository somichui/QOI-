## What is QOI+?

QOI+ is an improved version of the QOI image format. QOI was designed for simplicity and speed, deliberately leaving compression gains on the table. QOI+ reclaims those gains with three targeted improvements that have minimal performance cost on modern hardware.

### Improvements Over Vanilla QOI

| Feature | QOI | QOI+ | Impact |
|---------|-----|------|--------|
| **Pixel Prediction** | Left neighbor only | Paeth predictor (left, above, upper-left) | 15-25% smaller |
| **Hash Table** | 64 entries (256 bytes) | 256 entries (1 KB) | 5-10% fewer cache misses |
| **Entropy Coding** | None (fixed op-codes) | Optional Huffman coding | 10-20% additional reduction |
| **Run-Length** | Max 62 per op | Max 60 per op (2 slots for new ops) | Negligible tradeoff |

### Format Specification

```
Header (14 bytes):
  Bytes 0-3:   Magic "qoi+" (0x71 0x6F 0x69 0x2B)
  Bytes 4-7:   Width  (uint32, big-endian)
  Bytes 8-11:  Height (uint32, big-endian)
  Byte  12:    Channels (3 or 4)
  Byte  13:    Colorspace (0=sRGB, 1=linear)

Op-codes:
  00xxxxxx     INDEX       1 byte    6-bit hash index (0-63)
  01xxxxxx     DIFF        1 byte    2-bit dr, dg, db (-2..1)
  10xxxxxx     LUMA        2 bytes   6-bit dg, 4-bit dr-dg, 4-bit db-dg
  0xC0-0xFB    RUN         1 byte    Run length 1-60
  0xFC         INDEX_LONG  2 bytes   8-bit hash index (0-255)
  0xFE         RGB         4 bytes   Full r, g, b
  0xFF         RGBA        5 bytes   Full r, g, b, a

End marker: 8 bytes (0x00 × 7, 0x01)
```

### Paeth Predictor

Instead of comparing each pixel to its left neighbor (QOI), QOI+ uses the Paeth predictor from PNG:

```
predicted = paeth(left, above, upper_left)

paeth(a, b, c):
  p  = a + b - c
  pa = |p - a|
  pb = |p - b|
  pc = |p - c|
  return closest of a, b, c to p
```

This exploits 2D spatial correlation — the pixel above is often a better predictor than the pixel to the left, especially in photographs.

## Getting Started

### Prerequisites

- **Emscripten** (for WASM build): [Installation guide](https://emscripten.org/docs/getting_started/downloads.html)
- **Python 3** (for local server)
- **SDL2** (optional, for desktop app)

### Install Emscripten

```bash
git clone https://github.com/emscripten-core/emsdk.git
cd emsdk
emsdk install latest
emsdk activate latest
emsdk_env.bat        # Windows
source ./emsdk_env.sh  # macOS/Linux
```

### Build WASM

```bash
# From project root
build.bat            # Windows
# or manually:
emcc cpp/qoi_wasm.cpp -O3 --bind -o docs/qoi_core.js \
  -s WASM=1 -s ALLOW_MEMORY_GROWTH=1 -s MODULARIZE=1 \
  -s EXPORT_NAME=QoiModule -std=c++17 -I cpp/
```

### Run Locally

```bash
cd docs
python -m http.server 8080
# Open http://localhost:8080
```

### Build Desktop App (Optional)

```bash
g++ -o qoi_viewer cpp/main.cpp -lSDL2 -std=c++17
./qoi_viewer path/to/image.png
```
